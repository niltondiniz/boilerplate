# Boilerplate

  ### Running API
  To start API: docker-compose up -d
  
  ### Running Websockets
  1. Driver WebSocket: yarn start_driver
    Port: 8001
  2. Passenger WebSocket: yarn start_passenger  
    Port: 8002
   
  ### API Docs
  1. http://localhost:8000/doc

  ### API Status
  1. http://localhost:8000/health
